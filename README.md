## How to run it 
__Python__
1. Install Python and pip
2. `git clone https://github.com/vladyslavtarasov/Backend1`
3. `set FLASK_APP=backend_labs`
4. `flask run`

__Docker__
1. Install docker
2. `docker compose build`
3. `docker compose up`

__Render__

Open https://backend-labs-dxj1.onrender.com/
