## How to run it 
__Python__
1. Install Python and pip
2. `git clone https://github.com/vladyslavtarasov/Backend1`
3. `set FLASK_APP=lab1`
4. `flask run`

__Docker__
1. Install docker
2. `docker compose build`
3. `docker compose up`

__Heroku__

Open https://vladyslavtarasovlab1.herokuapp.com/
